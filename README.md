# pcc-data
アップロードされているFortranコードは「天体の位置計算 増補版」（地人書館 1998年第4刷 長沢 工著）に掲載されている計算方法をコード化したものです。
計算方法についての解説については同書を参照してください。

The uploaded Fortran code is a coded version of the calculation method published in "Augmented version of celestial position calculation" (Chijin Shokan, 1998, 4th edition, written by Ko Nagasawa). Please refer to the same book for the explanation of the calculation method.<br>
Sorry, Japanese Only

gfrotran [filename].f90 pcc_library.f90

./a.out (Linux)<br>
